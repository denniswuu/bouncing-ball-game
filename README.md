# 🎱 Bouncing Ball Adventure

弹球冒险小游戏 — 一个纯前端 HTML5 Canvas 弹球游戏！

## 🎮 游戏简介

点击屏幕发射弹球，收集彩色球获取分数，躲避紫色骷髅球！获得护盾和生命恢复道具来生存更久。

## ✨ 游戏特色

- **🎯 多彩收集品** — 收集不同颜色的球获取分数（10~30分不等）
- **💀 紫色骷髅敌人** — 躲避紫色骷髅球，碰到会扣除生命值
- **🛡️ 护盾道具** — 获得护盾可以消灭靠近的敌人
- **❤️ 生命恢复** — 收集红色爱心球恢复 30 点生命值
- **📈 难度递增** — 关卡越高，敌人越多、速度越快
- **🔥 Combo 连击** — 连续收集球获得额外分数加成
- **✨ 粒子特效** — 精美的爆炸粒子效果和星空背景
- **🎵 动感音乐** — 8-bit 风格背景音乐 + 丰富音效
- **📱 触屏支持** — 支持鼠标点击和触摸操作

## 🎵 音效系统

- **背景音乐** — 8-bit chiptune 风格旋律，带贝斯和鼓点节奏
- **收集音效** — 根据连击数音调逐渐升高
- **受伤音效** — 低沉的撞击声
- **护盾音效** — 上升的明亮音阶
- **发射音效** — 短促的加速声
- **游戏结束音效** — 下降的悲伤旋律

## 📝 游戏玩法提示

1. 优先收集高分球（绿色 30 分、金色 25 分）
2. 看到护盾球一定要捡，可以帮你消灭敌人
3. 生命值低于 50 时留意红色爱心球
4. 随着关卡提升，敌人数量和速度都会增加
5. Combo 连击越多，分数加成越高！
6. 紫色骷髅球是敌人，不是收集品，请避开它们！
