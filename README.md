# 🎱 Bouncing Ball Adventure

弹球冒险小游戏 — 一个纯前端 HTML5 Canvas 弹球游戏！

## 🎮 游戏简介

点击屏幕发射弹球，收集彩色球获取分数，躲避红色骷髅球！获得护盾和生命恢复道具来生存更久。

## ✨ 游戏特色

- **🎯 多彩收集品** — 收集不同颜色的球获取分数（10~30分不等）
- **💀 红色骷髅敌人** — 躲避红色骷髅球，碰到会扣除生命值
- **🛡️ 护盾道具** — 获得护盾可以消灭靠近的敌人
- **❤️ 生命恢复** — 收集红色爱心球恢复 30 点生命值
- **📈 难度递增** — 关卡越高，敌人越多、速度越快
- **🔥 Combo 连击** — 连续收集球获得额外分数加成
- **✨ 粒子特效** — 精美的爆炸粒子效果和星空背景
- **📱 触屏支持** — 支持鼠标点击和触摸操作

## 🚀 快速开始

### 方法一：直接在浏览器中打开

1. 下载本项目
2. 双击 index.html 文件，在浏览器中打开
3. 点击 "Start Game" 开始游戏！

### 方法二：使用本地服务器

`ash
# Python
python -m http.server 8080

# Node.js
npx serve .

# 然后用浏览器打开 http://localhost:8080
`

## 🛠️ 技术栈

- **HTML5** — 页面结构
- **CSS3** — 样式与动画
- **Vanilla JavaScript** — 游戏逻辑（Canvas API）
- **无任何第三方依赖** — 纯原生实现

## 📝 游戏玩法提示

1. 优先收集高分球（绿色 30 分、金色 25 分）
2. 看到护盾球一定要捡，可以帮你消灭敌人
3. 生命值低于 50 时留意红色爱心球
4. 随着关卡提升，敌人数量和速度都会增加
5. Combo 连击越多，分数加成越高！
